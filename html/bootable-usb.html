Format a drive to boot an ISO using MacOS CLI

Insert your USB stick into the mac and open Terminal. Wait a second, then run diskutil list

You'll see a list of all disks on your machine. Your USB stick will proabably be the last one. It's name will be something like /dev/diskN where "N" is an integer.

Run diskutil eraseDisk FAT32 NAME /dev/diskN

"NAME" can be anything but it must be all capital letters. This will erase the contents of your disk and reformat it to MS-DOS FAT 32, which is the format you want.

Run diskutil unmountDisk /dev/diskN

This will unmount your disk. Otherwise, when you try to write to it you will get a resource busy error.

Read the section below, then run sudo dd bs=1M if=/path/to/your/linux.iso of=/dev/rdiskN

Things to note:

  • dd is sometimes called "disk destroyer" because it sends the raw bytes of the file to the disk selected. Make sure you select the correct disk, or else you could end up destroying one of your drives, haha!

  • bs=1M defines the transfer rate. Sometimes you will get an error saying illegal numeric value. Just change it to 1m as opposed to 1M. It has to be capital or lowercase depending on whether you're using mac or linux and it's hard to keep it straight

  • if and of stand for input-file and output-file respectively

  • the "r" in rdiskN stands for raw and makes the process go faster, but you don't need it, and it could just as well be /dev/diskN

The computer will produce no visual output as it formats the file. You just have to wait. You can press ctrl-t though if you want an update. When it's done writing to the USB, your prompt will return and a window will pop up saying that the USB is unreadable by your computer. This is okay. You can choose to eject it from this pop up, or choose ignore and eject it from the command line by running diskutil eject /dev/diskN

That's it!
